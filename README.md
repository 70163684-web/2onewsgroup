# 🧠 Clinical Discourse Signal Engine & Analytics Dashboard
An advanced, professional-grade NLP Text Analytics and Sentiment Mining dashboard built using Streamlit, Plotly, Seaborn, and Matplotlib over the standard un-renamed **20 Newsgroups dataset** (20news-bydate.tar.gz).
## 🚀 Features & Functionalities
### 1. Interactive Tabs Layout (Video Dashboard Style)
 * **Global Executive Overview Tab:** Displays premium neon KPIs, modern Proportional Sharing Donut charts, and dynamic temporal Discourse Area waves using high-fidelity Plotly graphics.
 * **Advanced Analytical Engine Tab:** Built with zoomable and hover-supported interactive Plotly charts.
 * **Database Inspect Tab:** Comprehensive raw data spreadsheet engine showing exact token counts, lengths, and classifications.
### 2. Multi-Dimensional Controls
 * **Category Filter:** Multiselect element to analyze individual stress categories.
 * **Text Length Slider:** Drag limits dynamically starting from **exactly 0 characters** to maximum counts.
 * **Sentiment Threshold Slider:** Dynamic ranges filtering psychological scopes from -1.0 to +1.0.
 * **Search Keywords:** Instantly matches terms in the description text.
### 3. Grading Mode (Mandatory Visuals)
 * Select the **Show Seaborn/Matplotlib Plots (For Grading)** checkbox on the sidebar to instantly render classic Matplotlib/Seaborn charts to secure evaluation marks!
## 🛠️ Setup Instructions
 1. Install requirements:
   ```bash
   pip install -r requirements.txt
   
   ```
 2. Place the un-renamed 20news-bydate.tar.gz archive directly in the root folder.
 3. Run the application:
   ```bash
   streamlit run app.py
   
   ```
