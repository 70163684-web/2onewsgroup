# 20 Newsgroups Enterprise Visualizer Engine

A fully professional data visualization dashboard designed to extract, clean, and visualize statistical patterns from the classic **20 Newsgroups Text Corpus** dataset.

## Project Structure
- `app.py`: Main interactive UI controller framework.
- `data_processor.py`: Streaming backend regex engine parsing compressed binary tar files.
- `20news-bydate.tar.gz`: Core dataset untouched and unmodified to maintain submission compliance rules.
